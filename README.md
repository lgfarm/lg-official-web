# LG Farm Website Redesign

This project is a modern redesign of the LG Farm website using **React** and **Tailwind CSS**.

## Features
- Modern, responsive design.
- Glassmorphism effects in Navbar.
- Brand-aligned color palette (Orange & Dark Gray).
- Optimized Hero section with call-to-actions.

## Getting Started

### Prerequisites
Ensure you have Node.js installed.

### Installation
1.  Open the terminal in this directory.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Project
To start the development server:
```bash
npm run dev
```
Open your browser and navigate to the local URL (usually `http://localhost:5173`).

### Building for Production
To create a production build:
```bash
npm run build
```
The output will be in the `dist/` directory.
